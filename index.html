PERSON OF THE WEEK

Rick Viscomi (@rick_viscomi) is a Senior Web Developer Programs Engineer at Google. Passionate about web transparency, he maintains the Chrome User Experience Report and the open-source HTTP Archive project. To bring meaning to the raw stats and trends that the HTTP Archive has been collecting, Rick published the first issue of the Web Almanac this year, a yearly report on the state of the web, backed up by real data and trusted web experts. Rick is also the co-author of Using WebPage Test and host of the video series The State of the Web. Thank you for everything you do for the web community, dear Rick!
